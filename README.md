## 🎯 活动详情

### 🤿 Molokini 浮潜之旅
| 项目 | 详情 |
|------|------|
| 出发地 | Ma'alaea Harbor（距Kaanapali约35分钟） |
| 时长 | 约3-4小时（早班约7:00-11:00） |
| 价格 | 约 $100–170 / 人 |
| 亮点 | 夏威夷最顶级的浮潜地点，火山口内能见度极高，通常顺带游Turtle Town |
| 推荐运营商 | Four Winds、Malolo Charters、Pride of Maui |
| 预约 | 建议提前预订，选择早班避开风浪 |

### 🏔️ Haleakala 国家公园（七圣池 / Kīpahulu区）
| 项目 | 详情 |
|------|------|
| 地点 | Road to Hana 途中，Hana之后约10英里 |
| 入场费 | **$30 / 车**（有效期3天，含所有乘客） |
| 日出预约 | 需另外预约（$1/车），凌晨3:00–7:00进入必须预订，通过 recreation.gov 最多提前60天预约 |
| 官网 | https://www.nps.gov/hale/planyourvisit/fees.htm |
| 备注 | Road to Hana当天游Kīpahulu区（七圣池），无需日出预约 |
