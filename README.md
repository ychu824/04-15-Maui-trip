### 关于降雨概率

夏威夷的雨是热带阵雨，通常20–30分钟来了就走，和西雅图的连续阴雨完全不同。60%降雨概率不代表全天下雨，更多是"当天某个时段会有一阵雨"。对户外活动影响有限，Road to Hana下雨反而让瀑布和雨林更美。
